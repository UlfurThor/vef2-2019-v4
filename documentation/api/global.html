(async) get(req, res) → {Object}

Middleware that handles get requests to `./`. Can take 2 optional parameters through a json object: - `position`, controls if and how data should be ordered by the position field, takes in either `ascending` or `descending` as strings. - `completed`, controls if and how data should be filtered by the completed field, takes in Boolean, can returns for both true or false.
